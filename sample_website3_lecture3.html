Lecture 3 - Web Medium

Medium of the Web

Today's internet comprises of client/server models with three main elements:
The Server side
The Client side
The Network

Client/Server Model

Clients - request some type of service
Server - fulfills requests and transmits results to the client on the network Network - Describes the various connectivity elements used to deliver web sites to users

Server Types

A software installed on a computer Share resources Recieve requests and generate responses
  • Web servers
  • FTP(File Transfer Protocol) servers
  • Application server
  • E-mail server
  • Proxy server
  • Database server

    • HTTP protocol

    HTTP protocol is the main protocol used for communication between browsers and web servers
    Works by sending TCP connection to the web server. The browser sends request for a particular web page. The browser reads the content of the web page from the TCP connection and displays it. the browser closes the TCP connection used to transfer the web page.
    HTTP specifies commands that the client issues to the server and the responses that the server sends back to the client.

    Web architecture

    There are two types web architecture:
  • 2-Tier Architecture: Client and database
  • 3-Tier Architecture: Client, web service/processing, database

    • Uniform Resource Locator(URL)

    A formatted text string is used by web browsers to identify a resource on the network
    IP address and domain name identify each computer on the network. However, they can not indicate where the web pages' HTML
    To find it, web browsers depend on URLs, URL is a four part addressing scheme that tells the browser:
  • What transfer protocol to use in transferring the files
  • The doman name of the computer in which the files reside
  • The pathname of the folder or directory where the files reside
  • The name of the file

    • Domain name and Hosting

    A domain name is a unique name of an organisation or person on the internet. It used to identify computers on the internet. for example: nileuniverity.edu.ng
    Domain names can be bought from the domain registrar ICANN(International Commission for Assigned Names and Numbers)
    Web hosting, is a service that provides technologies and services needed for a website or web page to be viewed on the internet.

    What is a Website?

    A set of web pages served from a single web domain hosted, accessed pubicly through the network, identified by a unique address(URL), written in HTML and accessed through HTTP/HTTPS protocols
