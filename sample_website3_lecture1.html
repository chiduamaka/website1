Lecture 1 - Introduction to Web

What is the internet?

A networking infrastructure that allows devices to communicate with one another.

what is a protocol?

A set of rules that govern how transactions are carried out
Clients on the network request services from servers that provide these services

A Brief History of the internet

It Began in the late '60's in ARPANET, a US project investigating how to build networks to share information. Starting with a few nodes, the internet grew more and more with close to a billion users today

Internet Service models

The most common is the client/server model using appropriate protocols for communication. Ususally, local software clients access centralised remote servers.
Peer-to-peer model connects multiple, decentralised computers each acting as both clients and servers.
Clients and servers use protocols to provide services such as:
Web access
Email
File transfer
Data exchange
Chat/instant messaging
Shell/console access to computers
The most common protocols used today are; http/https, SMTP/MAP/POP, IP and many more

Protocols Notes
HTTP - Hypertext Transfer Protocol Core protocol for web transactions
SSL - Secure Socket Layer Data Encryption
HTTPS - Secure HTTP A combination of HTTP and SSL
SMTP - Simple Mail Transfer Protocol
POP - Post Office Protocol
IMAP - Internet Message Access Protocol
 Used in combination to provide most email services
FTP - File Transfer Protocol Used to exchange files between computers
SSH and SFTP - Secure Shell and Secure/SSH FTP Encrypted remote access and extension for encrypted file transfer
BitTorrent Peer-to-peer data exchange and communication

what is a domain name?

It makes navigating the internet easier by mapping IP addresses to text-based names.
example: 128.240.233.249 = www.netskills.ac.uk
When a device uses the domain name, it converted back to its numerical IP address by a DNS(Domain Name Server)
The web can be viewed as a system of interlinked hypertext document
HTML(Hypertext Markup Language) is the standard language for that defines a hypertext document

what are Cookies?
