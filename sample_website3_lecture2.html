Lecture 2 - Web Languages

The World Wide Web(WWW)

A system of interlinked hypertext documents accessible via the internet. They can be accessed using a web browser. These documents can contain multimedia files, video, audio and images
It was conceived by Timothy Berners-Lee and Robert Cailliau. It was launched at CERN in 1991 with HTML being the language to represent resources and HTTP, the protocol for transport of HTML files.

Web Languages

There are many web languages but we shall consider 6 of them: HTML, XML, XHTML, JSON, Javascript, and PHP

HyperText Markup Languages

It is designed to specify the structure of information presented on web pages

Elements and Tags

These are the logical units of the document. They start with "<" and end with ">"
Whitespace characters are ignored

Element Name Description
h1, h2, h3, h4, h5, h6 Headings
p Paragraph
table Table
tr Table row
ol Ordered list
ul Unordered List
li list item
form form to fill
hr horizontal rule
pre preformated text
div division
